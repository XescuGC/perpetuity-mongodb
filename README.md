# Perpetuity::MongoDB

This is the MongoDB adapter for Perpetuity.

## Installation

Add this line to your application's Gemfile:

    gem 'perpetuity-mongodb'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install perpetuity-mongodb
